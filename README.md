Title:
In-silico PCR Primer Design for Cancer Genes (TP53 & BRCA1)

Project description

This project documents a training workflow for designing and validating PCR primers for cancer research using bioinformatics tools.

The objective is to simulate real research workflows used in molecular oncology laboratories.

Skills demonstrated

Retrieval of genomic sequences from NCBI
PCR primer design using Primer3
Primer quality assessment (GC%, Tm, structure)
Primer specificity validation using NCBI BLAST
Scientific reasoning and troubleshooting mindset

Genes analyzed

TP53
BRCA1

Tools used

NCBI Nucleotide
Primer3
NCBI BLAST

Author

Master’s graduate in Biomedical Sciences – Molecular Oncology track.
